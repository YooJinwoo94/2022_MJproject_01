현재까지 작업 내역 :                                                   
플레이 팹에 사용자 등록을 하고 로그인을 한다.                                                  
로그인 후 현재 등록 했던 캐릭터를 불러 온다.                                                  
불러온 캐릭터를 unity상에 띄워 준다.                                                   
캐릭터의 ai는 behavior designer 와 fsm을 사용하였습니다.                                                   
( 전진 -> 적을 발견 -> 적으로 돌진 -> 사정거리 이내에 적이 있으면 근처에 부쉬가 있는지 검사 -> 만약 있으면 부쉬에 숨음 -> 전투 )
